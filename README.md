## Job Application Tracker (Assignment)

Simple job tracker assignment built with HTML, Tailwind CSS, and JavaScript. Users can move job cards between All, Interview, and Rejected, with counts and empty-state messages updating automatically.

### Features

- Move jobs to Interview or Rejected
- Live counts for total, interview, and rejected
- Empty-state UI when a section has no jobs
- Delete a job card

### Tech Stack

- HTML
- Tailwind CSS (CDN)
- JavaScript

### Assignment Notes

This project is a learning assignment focused on DOM manipulation and basic UI interactions.
